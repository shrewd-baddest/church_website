# CHURCH APPLICATION

## LOGIN PAGE

### DESCRIPTION

- The user tries to log in
- if the user forgets the password he/she clicks on reset password
- if the user's email is not recorded in the database, the user enters email,userName,new password and confirms
- if the user already has his/her email already recorded he is not required to fill the email again
- After filling all the fields an OTP is generated and sent to the user's email.
- The user is then directed to enter the OTP sent to the email to confirm the email really belongs to the user.
- If the user enter the right OTP he is directed back to the log in page else he will receive an alert to enter the correct OTP

### FEATURES

1.  LOGIN PAGE
2.  RESET PAGE
3.  OTP PAGE

### TECHNOLOGIES USED

2.  TAILWIND CSS

2. Run _**npm run dev**_ to install all the node modules from package json
3. start the server first with _**npm start**_/_**npm run dev**_
4. start the vite app with _**npm run dev**_
5. Now open the project from any browser with <http://localhost:5173/>
